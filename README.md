SeekYouRS.Examples
==========================

Examples for using the great [SeekYouRS](https://github.com/jfellien/SeekYouRS) framework with other storage options for models and aggregate event data.

Available storage options:
* MongoDB
* Need a different one? Open an issue and I will do my best :)

These examples are compatible with version **1.1.17.x** of SeekYouRS.
